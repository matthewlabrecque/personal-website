Projects

Playcaller

Done as part of a research project on the concept of "Disposable Development Environments", Playcaller takes the ideas behind an autoconfiguration script and pushes it to the max. This utility uses Ansible as a backend for creating automation scripts for the configuration and set up of any computer running a UNIX-based operating system.

Skills used: Ansible (Framework), Python

qSypher

The Sypher Foundry is a research project I am collaborating with Martin Burns on where we are studying and developing ciphers and hashing algorithms for quantum-level encryption. As part of this research project I am writing qSypher (quantumSypher), a memory-safe cryptographic tool written in Rust and Julia to test our ciphers.

Skills used: Rust, Julia, Cryptography

ET Runtime Telemetry Suite

The ET (Efficient Tensor) Platform/Runtime is part of an inititiave by AI-Foundry to create an Open-Source software/hardware stack for powering AI systems on the RISC-V architecture, specifically through serving as the bridge between the model and the hardware. I serve as part of the AI-Foundry team developing this software suite.

Skills used: C, Systems-Programming
